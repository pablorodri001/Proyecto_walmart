# Walmart Store Sales Forecasting – Data Project

## Descripción del proyecto
Este proyecto tiene como objetivo desarrollar una solución completa de **Data Engineering, Data Science y Machine Learning** para analizar y predecir las ventas semanales de Walmart, utilizando datos históricos reales.

El proyecto simula un **caso de negocio real**, donde las predicciones de ventas se utilizan para apoyar la toma de decisiones estratégicas como:
- planificación de inventario
- evaluación de promociones
- previsión de ingresos
- optimización operativa por tienda y departamento

---

## Objetivo de negocio
Predecir las **ventas semanales por tienda y departamento** con el fin de:
- anticipar demanda futura
- reducir roturas de stock
- mejorar la planificación financiera
- evaluar el impacto de factores externos (festivos, inflación, combustible, etc.)

---

## Dataset
Fuente: **Kaggle – Walmart Store Sales Forecasting**

Archivos principales:
- `train.csv`: ventas históricas
- `features.csv`: variables externas (CPI, Fuel Price, Temperature, etc.)
- `stores.csv`: información de tiendas
